# Twitter/X Timeline Sync

Userscript für Tampermonkey — Leseposition auf X/Twitter synchronisieren.

## Quelle

**GitHub (einzige Spiegelung):** https://github.com/Copiis/x-timeline-sync

Raw-URL (GreasyFork Code-Synchronisation + direkte Installation):

```
https://raw.githubusercontent.com/Copiis/x-timeline-sync/master/Twitter-X-Timeline-Sync.js
```

## Entwicklung

- Arbeitsdatei und Repo-Datei: `Twitter-X-Timeline-Sync.js` (eine Datei)
- Nach Änderung: `@version` erhöhen → `git add` → `commit` → `push`
- Tampermonkey lokal: `./update-tampermonkey.sh` → Zwischenablage einfügen
- GreasyFork holt Updates von der Raw-URL oben
