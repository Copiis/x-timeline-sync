X Reading Position & Media Download

Never lose your reading position again.

A Tampermonkey script that tracks and restores your place in the X/Twitter home timeline — automatically or manually. Works after reloads, "Show new posts", deleted tweets, and heavily changed feeds.

Highlights

  • Resolve engine — exact → context → trail → temporal → emergency fallback
  • New posts handling — loads new tweets at the top and restores your bookmark
  • Timeline map — remembers feed order for faster search
  • Repost-safe — separate handling with readAt timestamps
  • Visual marker — subtle 1px orange outline on the reading position
  • JSON export/import — history, bookmark, repost log
  • Media download — save images (original) and videos (best MP4) from tweets
  • 12+ languages — auto-detected UI

Media download

Hover a tweet image or video to show a download button (top-left corner):

  • Images — original resolution with automatic fallback
  • Videos — highest available MP4 quality
  • Works on all x.com pages with media (timeline, profile, single tweet)

Privacy

All data stays local in your browser. No server uploads. Reading-position sync runs on x.com/home; media download on x.com.
